# C++ Advanced Modules for the CPP Exam
The following modules prepare you for the [Certified Professional Programmer (CPP)](https://cppinstitute.org/cpp) exam. You can study for this exam by taking free courses available through the [Cisco Networking Academy](https://www.netacad.com/) website.

## C++ Advanced - CPP (Certified Professional Programmer)
- Module 1: STL Sequential Containers
- Module 2: STL Associative Containers
- Module 3: Non-modifying STL Algorithms
- Module 4: Modifying STL Algorithms
- Module 5: Sorting STL Operations
- Module 6: STL Merge Operations
- Module 7: STL Utilities and Functional Library
- Module 8: STL Advanced I/O
- Module 9: Templates
